Getting started

  • You can view the course content on nbviewer to get a feel for what's in the workshop.
  • If you want to go through the course material on your own PC, see the installation instructions for information on how to set up a Python environment with the appropriate packages.
  • You can also run Python exercises using a temporary session on binder: . It will let you test out the Python notebooks without downloading anything to your local machine.
